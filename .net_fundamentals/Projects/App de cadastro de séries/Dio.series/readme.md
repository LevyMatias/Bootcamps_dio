## App Cadastro de Series

Projeto Prático da Digital Innovation One - Criando um APP Simples de cadastro de séries em .NET 

Bootcamp .Net Fundamentals

### Funcionalidades

-  Listar series

-  Adicionar series

-  Excluir series

-  Atualizar series

-  Visualizar series

-  Sair

  

### Tecnologias ultilizadas

- .NET 5

- C#

- Visual Studio
